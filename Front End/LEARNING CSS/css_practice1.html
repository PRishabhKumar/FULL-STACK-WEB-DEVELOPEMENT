NOW WE WILL SEE ABOUT THE CHILD COMBINATOR



CHILD COMBINATORS ARE USED TO ESTABLISH THE CHILD RELATIONSHIP BETWEEN TWO ELEMENTS. FOR EXAMPLE IF WE WISH TO APPLY CERTAINSTYLING TO ALL TEH PARAGRPAHS INSIDE OF A DIV THEN I CAN USE THE CHILD COMBINATOR. THE DIFFERENCE BETWEEN THE CHILD COMBINATOR AND THE DESCENDANT SELECTOR IS THAT IN THE CASE OF DESCENDANT SECLCTOR, ALL THE ELEMENTS LYING UNDER THE PARENT ELEMENT, REGARDLESS OF WHAT LEVEL, WILL GET THE STYLING AND IN CASE OF THE CHILD COMBINATOR, ONLY THOSE SPECIFIED ELEMENTS WILL GET THE STYLING WHICH ARE DIRECTLY UNDER THE PARENT ELEMENT.
FOR EXAMPLE, IF I WANT TO APPLY SOME STYLING TO ALL PAATGRPAHS INSDE OF A DIV, IN THIS CASE IF I USE THE DESCENDANT SELECTOR, THEN IF I HAVE, SAY ANOTHER DIV INSIDE OF A PARENT DIV AND A PARAGRAPH INSIDE OF THAT DIV OR AN UNORDERED LIST, WHOSE LIST ELEMENTS ARE PARAGRAPHS, THEN ALL SUCH PARAGRAPHS WILL THE STYLING AS ALL OF THEM ARE UNDER THE PARENT DIV; BUT IN THE CASE OF TEH CHILD COMBINATORM, ONLY THOSE PARAGRPAHS WILL GET THE STYLING WHICH LIE DIRECLTY UNDER THE PARENT DIV.
